Q9. Linear Search Algorithm‡•§
Step 1: Start
Step 2: Set i = 0
Step 3: If array[i] == Item then Print "Found" and Stop
Step 4: i = i + 1
Step 5: If i < n repeat Step 3, else Print "Not Found"
Step 6: Stop
Q10. Binary Search Algorithm‡•§
Step 1: Start | low = 0, high = n-1
Step 2: mid = (low + high) / 2
Step 3: If Item == array[mid] then "Found" and Stop
Step 4: If Item < array[mid] then high = mid - 1
Step 5: Else low = mid + 1
Step 6: If low <= high repeat Step 2, else "Not Found"
